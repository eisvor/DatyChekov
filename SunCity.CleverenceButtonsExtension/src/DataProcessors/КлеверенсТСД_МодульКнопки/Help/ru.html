Внешняя обработка предназначена для настройки и выполнения обмена данными с мобильными устройствами, работающими под управлением программного обеспечения на платформе Mobile Smarts (разработчик - компания "Клеверенс").

Запуск внешней обработки возможен из файла (Файл - Открыть) либо из справочника дополнительных отчетов и обработок.

Торговый объект 
Торговый объект - это условная структурная единица (склад, магазин и т.д.), предназначенная для разграничения настроек обмена с различными мобильными устройствами. В поле отображается торговый объект, с которым происходит работа в настоящий момент. Можно просмотреть параметры торгового объекта, выбрать другой торговый объект или очистить его.

Выгрузить справочники
По нажатию на кнопку начинается выгрузка справочников на мобильное устройство.

Открыть окно обмена справочников

По нажатию на кнопку открывается список справочников, из которого можно выбрать те, которые необходимо выгружать, и указать их настройки.

Открыть окно обмена документами

По нажатию на кнопку открывается окно обмена документами. В окне обмена документами можно выгрузить документы на мобильное устройство, загрузить их с мобильного устройства и просмотреть, а также завести неизвестные штрихкоды и распечатать ценники.

Показать настройки

По нажатию на кнопку открываются настройки обработки:

  • Подключенная база Mobile SMARTS - база Mobile SMARTS, с которой происходит работа в настоящий момент. По кнопке открытия можно указать или просмотреть параметры подключения к базе.
  • Настройка бизнес-процессов - кнопка открытия окна настройки бизнес-процессов, в котором можно просмотреть или отредактировать настройки обмена в разрезе различных товароучетных операций.
  • Настройка ручного обмена - по нажатию на кнопку открывается окно настройки ручного обмена.
  • Настройка онлайн-обмена - по нажатию на кнопку открывается окно настройки онлайн-обмена.
  • Настройка торговых объектов - кнопка открытия окна настройки торговых объектов.
  • Настройка глобальных параметров - кнопка открытия окна настройки глобальных параметров обмена. Есть возможность указать индивидуальные значения глобальных параметров для разных торговых объектов. 
  • Расширенные настройки - по нажатию на кнопку открываются дополнительные настройки работы обработки с базой Mobile SMARTS.

    "Информация"


  • Подключенная база Mobile SMARTS - база Mobile SMARTS, с которой происходит работа в настоящий момент.
  • Версия основной обработки - системные данные обработки, версия, дата выпуска и пр. 
  • Текущая конфигурация - конфигурация 1С, с которой происходит работа в настоящий момент. 
  • Версия обработки интеграции - версия подключенной интеграционной обработки. 
  • Путь к обработке интеграции - путь к подключенной интеграционной обработке.
  • Конфигурации 1С, поддерживаемые обработкой интеграции - конфигурации 1С, с которыми возможна работа интеграционной обработки. 
  • Версии конфигураций 1С, поддерживаемые обработкой интеграции - диапазон версий конфигураций, поддерживаемых интеграционной обработкой. 
  • Состояние - информация о текущем состоянии интеграционной обработки, пригодности ее к работе или наличии ошибки, не позволяющей приступить к работе.
