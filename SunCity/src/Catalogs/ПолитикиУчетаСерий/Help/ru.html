Политики учета серий

Справочник служит для формирования различных правил ведения учета серий и сроков годности номенклатуры. Использование серий для номенклатуры задается в виде номенклатуры. Правила должны быть сформированы для всех видов номенклатуры, по которым необходимо вести учет серий или сроков годности. Информация о правилах учета серий для вида номенклатуры может быть детализирована для каждого склада.

Возможность использовать серии при учете движения товаров управляется настройкой по ссылке Администрирование – Настройки номенклатуры – Общие параметры - Серии товаров.

Информация о том, какая политика учета серий применяется на конкретном складе задается в карточке вида номенклатуры.

Список элементов открывается по ссылке НСИ – Номенклатура – Политики учета серий. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Пример. На предприятии ведется контроль серийных номеров бытовой техники при реализации, чтобы можно было контролировать их при возврате или приемке в сервис для ремонта. В этом случае для вида номенклатуры «Бытовая техника» нужно указать, что серия идентифицирует экземпляры товаров, имеющих уникальные серийные номера, а в политике учета серий нужно указать:при отгрузке клиенту, в розницу и при приемке – по возврату от клиента. В этом случае в документах Реализация товаров, Чек ККМ, Отчет о розничных продажах, Возврат от поставщика нужно будет обязательно вводить серийный номер для номенклатуры данного вида.
