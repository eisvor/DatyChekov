Серийные номера

Серийные номера используются для идентификации серии вида товаров:

· экземпляры товаров, имеющих уникальные серийные номера, например, для бытовой техники и электроники;

· партии товаров, имеющие одинаковые номера серий, – для товаров серийного производства, например, лекарственных препаратов;

· партии товаров, имеющие одинаковые сроки годности, – для товаров, имеющих определенную дату окончания реализации;

· партии товаров, имеющие одинаковые номера серий и сроки годности, – для учета и номера серии, и даты окончания реализации.

Для каждого магазина сети может быть выбрана своя политика учета серий в документах товародвижения. В политике учета серий указывается, в каких документах при формировании табличной части нужно будет указать серии товаров.

Внесение серийного номера подарочного сертификата в документ

Если сертификаты имеют серийные номера, в документе можно указать диапазон серийных номеров воспользовавшись командой Ввести серийные номера в командной панели табличной части документа. В открывшейся форме доступен выбор диапазона из числа существующих или предоставляется возможность создать новый диапазон серийных номеров подарочных сертификатов.
