Регистрационные карты

В справочнике хранится информация о регистрационных картах сотрудников и дисконтных картах покупателей.

Информация с карты может быть получена считывателем магнитных карт или сканером штрихкодов. Для тех карт, которые считываются считывателем магнитных карт, устанавливается вид карты – магнитная, а если информация считывается сканером штрихкодов, то устанавливается вид карты – штриховая. Для тех карт, информация с которых считывается со сканера, из справочника "Типы штрихкодов" выбирается тип штрихкода.

Список карт открывается по ссылке Персонал – Сотрудники – Регистрационные карты (Маркетинг – Маркетинговые акции – Дисконтные карты). С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. С помощью кнопки Создать новую группу Пользователь может добавить иерархию в дерево регистрационных карт.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

  • В поле Наименование указывается краткое наименование дисконтной карты.
  • В поле Код карты вводится код дисконтной карты.
  • В поле Владелец карты можно указать контрагента, которому выдана карта.
  • Возможность работы с дисконтными картами покупателей определяется в настройках учетной политики .

Если тип карты - регистрационная, то Владельцем карты может быть либо пользователь, либо физическое лицо, который является сотрудником (установлен флаг Является сотрудником).

Дисконтные карты предъявляются покупателем при оплате. В соответствии с предъявленной дисконтной картой покупателю может быть предоставлена скидка. Условием предоставления скидки может быть, как конкретная дисконтная карта, так и карты определенного вида, например, "Серебряная", "Золотая", "Платиновая". Вид дисконтной карты задается в поле Вид диск. карты выбором элемента справочника Виды дисконтных карт.

При выдаче дисконтной карты покупателю можно настроить автоматическое проведение опроса. При этом запись в справочнике физических лиц - владельцев карт будет создаваться автоматически. Для этого в форму опроса добавлены обязательные поля: ФИО, пол, дата рождения, адрес фактический, телефон и адрес электронной почты.

Если данные о владельце карты уже введены в систему, они будут отображены в форме опроса. Их можно отредактировать, обновив данные на текущий момент.

Опрос покупателей, имеющих дисконтные карты

Сбор и хранение в системе данных о покупателях имеет своей целью создание профиля покупателя, являющегося типичным для данного предприятия.

На закладке Регистрация, опрос вносится дата регистрации карты либо вручную, либо с помощью механизма автоматической регистрации дисконтной карты в режиме РМК. Для активизации механизма необходимо установить флаг Проводить опрос владельца карты на закладке Общие настройки в справочнике Настройки РМК. и задать шаблон кода для регистрации новых карт в регистре сведений. Карта регистрируется в системе только в том случае, если ее код подходит под шаблон.

После регистрации карты система предлагает ввести ответы покупателя:стандартные реквизиты справочника Физические лица: фамилия, имя, отчество покупателя, его пол и дата рождения, а также дополнительные реквизиты и сведения, созданные пользователем.

Пользователь может также установить дату следующего опроса, которая активизирует запрос кассиру на ввод данных в режиме РМК при считывании его дисконтной карты. Дата устанавливается либо вручную, либо с помощью обработки Заполнение дат следующего опроса из формы списка справочника.
