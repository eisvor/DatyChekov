Аналитика хозяйственных операций

Справочник предназначен для расширения аналитик учета хозяйственных операций. Аналитика учета хозяйственных операций может быть указана в документах Оприходование товаров, Списание товаров, Возврат товаров от покупателя, Возврат поставщику.

В справочнике имеются предустановленные элементы для аналитического учета  хозяйственных операций, которые можно использовать при оформлении документов. Пользователь может добавить любое количесво дополнительных аналитических признаков для следующих хозяйственных операций.

  • Возврат от покупателя
  • Возврат поставщику
  • Оприходование
  • Оприходование по инвентаризации
  • Списание на затраты
  • Списание по инвентаризации
  • Оприходование комиссионных товаров
  • Вознаграждение от комитента.

Список элементов открывается Закупки – См.также - Аналитика хозяйственных операций или Администрирование – Запасы и закупки – Аналитики хозяйственных операций. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов.

Новый аналитический признак можно добавить, нажав в командной панели кнопку Создать. В открывшейся форме ввести наименование аналитики и указать из перечисления хозяйственную операцию, к которой она относится.
