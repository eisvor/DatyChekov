Шаблон печати кодов маркировки

Справочник предназначен для хранения шаблонов печати кодов маркировки.

Поддерживается печать штрихкодов в двух форматах: GS1-DataMatrix и DataMatrix. Потребительская упаковка табачной продукции должна маркироваться кодом в формате DataMatrix. Все прочие коды маркировки генерируются в формате GS1-DataMatrix.

Штрихкод, размещаемый в шаблоне по умолчанию, генерируется в формате GS1-DataMatrix. Признак Используется для табачной пачки, выставленный у элемента справочника, позволяет изменить формат генерации штрихкода на DataMatrix.

Использовать шаблон с выставленным признаком Используется для табачной пачки при печати этикеток иных форм упаковок и товарных групп не рекомендуется, т.к. это будет нарушать правила маркировки продукции на территории РК.

Печать кодов маркировки с использованием созданных шаблонов возможна из Пула кодов и из рабочего места Печать загруженных кодов маркировки
