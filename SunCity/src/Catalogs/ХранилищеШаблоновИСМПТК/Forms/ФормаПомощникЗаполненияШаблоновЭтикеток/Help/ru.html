Помощник заполнения шаблонов печати

Помощник упрощает выбор шаблона печати этикеток кодов маркировки и позволяет назначить один шаблон для печати сразу нескольких позиций списка. Применятся в форме уточнения данных печати при распечатке кодов маркировки из Пула кодов и из рабочего места Печать загруженных кодов маркировки.

В поле Шаблон потребительской этикетки указывается макет печати, который нужно заполнить во всех строках списка данных печати.

В поле Количество экземпляров (шт) указывается количество кодов маркировки из каждой строки списка печати, которые нужно вывести на печать. Указанное количество будет установлено во всех строках в форме печати кодов маркировки. Если оно будет превышать количество доступных для печати кодов в строке, значение в ней изменено не будет.
