Шаблоны печати кодов маркировки

Справочник предназначен для хранения шаблонов печати кодов маркировки.

Поддерживается печать штрихкодов в двух форматах: GS1-DataMatrix и DataMatrix. Штрихкод, размещаемый в шаблоне по умолчанию, генерируется в формате GS1-DataMatrix. Для печати потребительских кодов табачной продукции следует использовать шаблон с выставленным признаком Используется для табачной пачки.

Печать кодов маркировки с использованием созданных шаблонов возможна из Пула кодов и из рабочего места Печать загруженных кодов маркировки.
