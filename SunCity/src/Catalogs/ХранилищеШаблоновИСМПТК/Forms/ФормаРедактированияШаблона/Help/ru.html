Редактирование шаблона печати кода маркировки

Редактор предназначен для создания макета печати этикетки кода маркировки.

Новый шаблон этикетки может быть создан полностью самостоятельно или на основании шаблона, предусмотренного по умолчанию. Также возможен выбор одного из стандартных шаблонов этикеток.

Для загрузки предопределенного шаблона необходимо воспользоваться командой Загрузить стандартный шаблон и выбрать подходящий макет этикетки из предложенного списка.

С помощью команды Импортировать можно загрузить макет этикетки из внешнего файла. Команда Экспортировать позволяет выгрузить сформированный макет во внешний файл.

Команда Установить область печати предназначена для указания прямоугольной области макета, которая будет выводиться на печать. Поля макета, не попадающие в эту область, при печати будут игнорироваться.

В группе Доступные поля расположены параметры, которые можно разместить в макете. При формировании этикетки во время печати эти параметры будут заполняться при наличии соответствующих сведений в данных печати.
