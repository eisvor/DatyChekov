Настройки РМК

Справочник Настройки РМК предназначен для сохранения различных настроек интерфейса рабочего места кассира.

Администратор может создать неограниченное количество различных настроек РМК. Ограничение функциональных возможностей кассира при работе в РМК достигается за счет сочетания настроек интерфейса РМК и персональных дополнительных прав пользователя.

Список дополнительных прав пользователя открывается по ссылке Администрирование – Настройки пользователей и прав - Персональные настройки пользователя - Дополнительные права пользователей. В режиме РМК можно без выхода из программы применить дополнительные права другого пользователя (например, старшего кассира) для выполнения запрещенных для кассира операций. Для этого нужно воспользоваться командой Заменить права.

Список элементов справочника Настройки РМК можно открыть по одноименной ссылке в панели навигации раздела Продажи - См.также или по ссылке Администрирование - Продажи – Рабочее место кассира (РМК) - Настройки РМК.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Наименование применяемой на данном компьютере настройки РМК выделено в списке шрифтом. Чтобы выбрать другую настройку, нужно выбрать ее в списке и нажать кнопку Применить настройку.

При создании новой записи Администратор вводит общее наименование настройки РМК и редактирует детальные настройки на страницах Общие настройки, Кнопки нижней панели, Кнопки верхней панели и Информация о текущей строке.

Общие настройки

Флажок Объединять позиции с одинаковым товаром устанавливается, если при повторном добавлении товара нужно изменять его количество. Если флажок не установлен, то информация о каждом введенном товаре будет добавлена в чек отдельной строкой.

Установка флажка Использовать расширенную настройку кнопок нижней панели позволяет добавить дополнительные кнопки управления  в нижней панели.

Установка флажка Открывать нижнюю панель при запуске позволяет автоматически открывать кнопки нижней панели при запуске РМК.

Установка флажка Назначать цену товарам с нулевой ценой означает, что цены на товар, если они не заданы заранее, могут быть введены кассиром при добавлении товара в чек в отдельном окне.

Установка флажка Запретить выбор дисконтной карты из списка закрывает доступ кассира к общему списку информационных карт сотрудников. Идентификация сотрудника производится при вводе кода его информационной карты без предоставления кассиру общего списка информационных карт всех сотрудников.

Установка флажка Выбор упаковки при продаже позволяет выбрать упаковку для товара, который продается в различных упаковках.

Флажок Закрывать окно поиска при выборе товара управляет поведением формы подбора товара из справочника Номенклатура.

Открывать редактирование строки при поиске товаров. При установленном флаге после выбора найденного товара открывается окно редактирования строки в нем можно изменить количество, серии, сертификаты, продавца и цену товара (в зависимости от дополнительных прав пользователя).

При установленном флажке Проводить опрос владельца карты при оформлении чека ККМ становится доступной регистрация дисконтных карт и ввод анкетных данных покупателя.

Установка флажка Контролировать остатки товаров при закрытии чека позволяет предотвратить продажи товара в количестве большем, чем учетные остатки по складу хранения. Проверка осуществляется при переходе к оплате чека и в момент проведения документа Чек ККМ.

Установка флажка При возврате распечатывать пакет документов позволяет кассиру при оформлении возвратов покупателю распечатать необходимые документы, сопровождающие возврат товаров: заявление покупателя или кассира о причине возврата, акт возврата, расходный кассовый ордер.

Установка флажка Всегда открывать форму множественного числа акцизных марок позволяет одновременно считывать несколько акцизных марок.

Установка флажка Копировать парметры бонусов при замене карты позволяет сохранять количесство накопленных бонусов при создании новой бонусной карты путем копирования старой.

Указание параметра Интервал автоматического сохранения чека ККМ позволяет сохранять информацию о текущем чеке во внешний файл через указанное пользователем количество секунд. Информация о чеке сохраняется до момента штатного завершения работы с чеком. Это обеспечивает быстрое восстановление работы кассира при форс-мажорных обстоятельствах, например, при сбоях в обеспечении электропитанием. При входе в режим регистрации продаж программа анализирует локальный файл и, если такой чек присутствует, восстанавливает его.

Флажок Предупреждение о закрытии смены за - "Время предупреждения о скором закрытии смены". При наступлении выдается сообщение о необходимости закрытия кассы. Сообщение остается видимым до закрытия РМК.

Пользователь может выбрать Шаблон работы дисплея покупателя из списка предварительно созданных шаблонов. Шаблоны могут различаться, например, для разных моделей используемых на предприятии дисплеев покупателей. Если шаблон работы дисплея покупателей не выбран, то на дисплей выводится информация, предустановленная разработчиком драйвера.

Кнопки нижней панели

Настройка команд нижней панели РМК производится на странице Кнопки нижней панели, которая становится доступна, если на странице Общие настройки установлен флажок Использовать расширенную настройку кнопок нижней панели. Настройка заключается в выборе необходимых для работы кассира команд из списка 29 предустановленных команд и настройке горячих клавиш вызова с клавиатуры. Список может быть заполнен по умолчанию всеми предустановленными командами с помощью кнопки Заполнить и отредактирован администратором.

Положением команды в списке можно управлять с помощью стрелок в командой панели списка. Панель дополнительных команд интерфейса РМК формируется в соответствии со списком: заполнение производится слева направо и сверху вниз. Таким образом, администратор может настроить панель так, чтобы кассиру удобно было ориентироваться в командах. Недоступные данному пользователю команды выводятся на командные панели интерфейса РМК, но выделены бледным шрифтом. Для команд можно использовать предпочитаемые шрифты и цвета. В нижней части формы есть возможность добавить палитру быстрых товаров через кнопку выбора.

Кнопки управления вводом информации в чек

· Быстрые товары (Alt + F2) – открыть/закрыть панель выбора быстрых товаров.

· Штрихкод (Alt + F7) – открыть форму ручного ввода штрихкода.

· Получить вес (Ctrl + F2) – получить вес из весов, подключенных к рабочему месту кассира.

· Продавец (Ctrl + F4) – ввести продавца при учете личных продаж. По этой команде на продавца относится весь чек. Убрать ссылку на продавца можно с помощью повторного нажатия кнопки. Указать продавца для одного из товаров чека можно с помощью сервиса редактирования строки, который вызывается кнопкой верхней панели Ред.Строки (F4).

· Информ. карта (Ctrl + F7) – открыть форму для ручного ввода кода информационной карты.

· Подарочн. сертификаты (Ctrl + F8) – открыть форму для ручного ввода серийного номера подарочного сертификата при его продаже, если по какой-либо причине нельзя считать штриховой или магнитный код сертификата в фоновом режиме.

· Получить из ТСД (Shift + F7) – загрузить из ТСД список предварительно отсканированных товаров, что позволяет ускорить оформление розничных продаж. При наличии очереди продавец может отсканировать товары на терминал сбора данных. Кассиру останется только загрузить список товаров, рассчитать скидки и принять оплату от покупателя.

· Ввод серий (Shift + F2) – открыть форму для ввода серии товара. Эта опция востребована, если в магазине ведется учет серий при продаже товаров какого-либо вида. Открыть форму для ввода серии можно также, нажав на значок серии, который отображается в зоне текущей строки рядом с наименованием номенклатуры;

· Продажа по заказу – открыть форму со списком заказов покупателей, которые были заранее сформированы и собраны на складе. Если кассир выберет заказ из списка, то в чек будут загружены все непроданные ранее товары выбранного заказа.

Кнопки управления скидками

· Расчет скидок (Alt + F9) – рассчитать скидки. При исполнении команды будут определены и рассчитаны автоматические скидки. Команда выполняется по умолчанию при переходе к оплате.

· Ручные скидки (Shift + F8) – пользователь, имеющий право предоставлять ручные скидки, с помощью этой кнопки может ввести значение скидки.

· Управл. скидки (Ctrl + F9) – назначить автоматическую скидку, выбрав из открывшегося списка управляемых вручную скидок.

· Замена карты (Ctrl + F11) – произвести замену дисконтных карт покупателя. При использовании накопительных скидок накопления переходят на новую карту.

Кнопки управления чеком

· Отложить текущ. чек (Alt + F5) – отложить оформление чека. Вернуться к оформлению этого чека можно позднее с помощью кнопки Продолж. чек (Ctrl + F5). Отложенные чеки сохраняются до закрытия смены и не используются для формирования документа Отчет о розничных продажах.

· Продолж. чек (Ctrl + F5) – продолжить отложенный ранее чек, выбрав его из списка отложенных чеков.

· Аннулир. чека (Alt + F6) – аннулирование текущего чека. При использовании этой команды текущему чеку присваивается статус Аннулирован. Аннулированные чеки не используются для формирования документа Отчет о розничных продажах, но сохраняются в базе данных для анализа.

Кнопки управления печатью

· Пробить непробитый (Ctrl + F12) – используется при ошибке печатающего устройства. После устранения неполадки с помощью этой команды можно пробить чек, на котором прервалась печать.

· Печать пробитого чека (Shift + F11) – данная команда позволяет выбрать печатную форму, по которой пробитый чек текущей смены будет выведен на офисный или узколенточный принтер. В случае фискального регистратора печать производится в нефискальном режиме.

· Печать (Alt + F11) – печать дополнительных форм текущего чека.

· Послед. слип чек (Alt + F12) – по этой команде напечатается повторно последний слип-чек оплаты по платежной карте.

· Отчет без гашения (Shift + F12) – команда позволяет сформировать, просмотреть и распечатать Х-отчет по текущей смене.

Другие операции

· Заменить права (Alt + F4) – заменить права текущего сотрудника (кассира) на права другого сотрудника (например, старшего кассира), чья карточка будет считана после нажатия кнопки. При этом заменяются только дополнительные права пользователя. Для возврата прав кассира нужно повторно нажать на кнопку и считать карточку текущего кассира.

· Открыть ДЯ (Alt + F8) – открыть денежный ящик. Кнопка действует, если команда на открытие денежного ящика поддерживается соответствующим драйвером.

· Внесение денег (Alt + F3) – открыть форму, в которой отражаются предварительно оформленные в операционной кассе магазина документы выдачи ДС на кассы ККМ данного рабочего места. Кассиру необходимо выбрать в списке нужный документ и зачислить сумму размена на счет ККМ.

· Выемка денег (Ctrl + F3) – выполнить выемку денежных средств в течение смены. Кассир может последовательно оформить выемку денежных средств для каждой кассы ККМ данного рабочего места, при этом автоматически формируются документы Выемка денежных средств из кассы ККМ. При закрытии смены выемка остатка денежных средств формируется автоматически.

· Возврат покупателя (Ctrl + F6) – оформить возврат товаров от покупателя. Вызывается сервис для формирования документа Возврат товаров от покупателя. Данная опция используется для оформления возврата товаров, которые были проданы в течение уже закрытых кассовых смен. Для оформления возврата товаров, проданных в текущей смене, используется команда Возврат (F5).

Выход

· Выход (F12) – выход из режима РМК. Наличие этой кнопки в нижнем меню обязательно. Эту команду нельзя удалить из списка кнопок нижнего меню при настройке.

Кнопки верхней панели

· Меню (F10) – выход в основное меню.

· Поиск (F11) – открыть форму подбора товаров для поиска нужного товара.

· Возврат(F5) – оформить возврат товаров, проданных в текущей смене. .

· Без передачи товаров (Shift + F10) – оформить предоплату без передачи товаров. Предоплату можно оформить платежной картой или наличными.

· Бонусы (Shift + F9) – оформить оплату бонусами.

· Наличные (F6) – оформить оплату наличными.

· Плат.карта (F7) – оформить оплату платежной картой

· Сложная оплата (F8) – оформить смешанную оплату доступными средствами: наличными, платежной картой, бонусами. 

Информация о текущей строке

Алгоритм действует следующим образом. Если настройка РМК не применена для данного компьютера или если альтернативный механизм не задан, то вывод в табло РМК будет осуществляться прежним способом. А именно для номенклатуры будет выводиться только наименование. Если альтернативный механизм будет задан, то будет использован он.

Предварительный просмотр табло реализован в закладке Информация о текущей строке. Информация может быть задана путем выбора реквизитов номенклатуры из списка или создана с использованием конструктора. Используя конструктор Пользователь может реализовать любой запрос для получения дополнительной информации по номенклатуре (свойства, цены, остатки и т.п.).
