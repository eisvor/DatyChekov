Номера ГТД

В этом справочнике хранятся номера всех грузовых таможенных деклараций, по которым когда-либо поступали импортные товары.

Список элементов открывается по ссылке НСИ – Номенклатура – Номера ГТД. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка. Справочник имеет одноуровневую структуру.
