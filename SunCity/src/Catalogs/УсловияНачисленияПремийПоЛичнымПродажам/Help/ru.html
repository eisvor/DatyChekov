Условия начисления премий поличным продажам

Мотивация продавцов - один из действенных способов увеличить объемы продаж в организации. В конфигурации реализован функционал, позволяющий рассчитывать премии сотрудникам на основании данных их личных продаж. Правила премирования вносятся в справочник Правила расчета премии и Условия расчета премий.

Список условий открывается по ссылке Персонал – Премирование – Условия начисления премий. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

С помощью кнопки Создать группу Пользователь может добавить иерархию в дерево Условий.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.
