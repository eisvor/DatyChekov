Бонусные программы лояльности

Справочник Бонусные программы лояльности предназначен для описания параметров бонусных программ и ограничений при оплате бонусными баллами по этим программам.

Бонусные программы - это вид автоматических скидок, при котором скидка в денежном выражении не предоставляется покупателю немедленно, а накапливается в виде бонусов на особом счете покупателя, связанном с его дисконтной картой. Возможность использования бонусных программ управляется Администратором с помощью флага Бонусные программы лояльности по ссылке Администрирование - Маркетинг. Правила начисления бонусных баллов описываются в справочнике Скидки. Бонусные программы вводятся в действие документом Маркетинговые акции, так же, как и обычные скидки. Для каждой бонусной программы можно также настроить автоматическое начисление и списание бонусов, не зависящие от покупки с помощью справочника Правила расчета бонусных баллов. Программа предоставляет возможность проверки состояния бонусного счета покупателя при предъявлении карты и вывода в чек параметров бонусной программы.

Список элементов открывается по ссылке Маркетинг – Бонусные программы – Бонусные программы. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные программы лояльности, задавая поиск по значениям различных реквизитов.

Создать новую бонусную программу можно с помощью кнопки Создать командной панели справочника. При создании новой бонусной программы Пользователь вводит наименование программы и курс пересчета бонусных баллов в тенге (при оплате покупки баллами) в поле Один бонусный балл равен.

Пользователь может установить режим начисления бонусных баллов при оплате покупки баллами: баллы могут начисляться как при обычной оплате, а могут не начисляться.

Пользователь может выбрать определенный сегмент номенклатуры, которую можно оплатить баллами. Если сегмент не выбран, то любой товар может быть оплачен баллами. Поскольку сумма, оплаченная баллами, при передаче сведений в бухгалтерию вычитается из выручки, Пользователь должен установить максимально возможный процент оплаты покупки баллами. Этот процент Пользователь может уточнить для разных ценовых групп в табличной части карточки бонусной программы. При выборе максимальной доли оплаты рекомендуется учитывать правила расчета розничных цен, установленных для участвующих в программе товаров.
