Ценовые группы

Справочник Ценовые группы служит для создания групп, объединяющих товары с одинаковыми правилами расчета цен. Для использования ценовых групп следует установить параметр Ценовые группы (Администрирование – Маркетинг - Ценообразование).

Список групп открывается Маркетинг – Ценообразование – Ценовые группы. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные группы, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка. При создании нового элемента справочника пользователь должен задать название ценовой группы, достаточно ясно отражающей ее содержание.

Пример. Номенклатура с ценовой группой "Наценка 20% от цены поставщика" рассчитывается на основании цены поставщика путем умножения на коэффициент к=1.2.
