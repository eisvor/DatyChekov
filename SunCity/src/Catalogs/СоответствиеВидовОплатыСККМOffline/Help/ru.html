Соответствие видов оплаты с ККМ offline

Это правила подстановки вида оплаты при загрузке данных из ККМ Offline.

Список правил открывается по ссылке Администрирование – Подключаемое оборудование - Соответствие видов оплаты с ККМ Offline. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новое правило/соответствие создается с помощью кнопки Создать. Далее в левой части надо указать код оплаты на ККМ Offline, а справа выбрать Вид оплаты. Команда Добавить добавляет одну строку для соответствия, Добавить все виды оплат – все виды оплат.
