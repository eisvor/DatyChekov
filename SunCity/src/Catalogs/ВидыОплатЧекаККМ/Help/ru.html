Виды оплат

Настройки Виды оплат документа Чек ККМ содержат предустановленные виды оплат Наличные, Оплата подарочным сертификатом. Пользователь может настроить дополнительно виды оплат платежной картой или банковским кредитом.

Список оплат чеков ККМ можно открыть по ссылке Администрирование - Продажи – Оплата - Виды оплаты. В список выводится Наименование, Тип оплаты и Процент комиссии банка для типов оплат "Банковский кредит" и "Платежная карта". С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные оплаты, задавая поиск по значениям различных реквизитов.

Справочник содержит два предопределенных элемента:

  • Наличные - оплата наличными поддерживается программой при любых настройках;

  • Оплата подарочным сертификатом - становится доступной, если в программу введен хотя бы один подарочный сертификат.

При оплате подарочным сертификатом в РМК номер сертификата можно ввести с помощью сканера штрихкодов или считывателя магнитной карты.

Виды оплат с типами Банковский кредит и Платежная карта вводятся пользователем по мере необходимости.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Чтобы настроить оплату чеков ККМ платежными картами нужно вести в программу информацию о поддерживаемых банком-эквайрером видах платежных карт.
