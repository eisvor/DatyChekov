Кассы ККМ

В справочник Кассы ККМ вводится информация о контрольно-кассовых машинах предприятия. Кассы ККМ предназначены для регистрации денежных средств, полученных от розничного покупателя. При продаже потоковым методом кассы ККМ связываются с рабочим местом кассира (компьютером). В этом случае кассир работает с кассой ККМ через интерфейс РМК, настройки которого хранятся в справочнике Настройки РМК. Если на одной точке продаются товары (услуги) от имени нескольких организаций, то этот процесс можно организовать через одно рабочее место кассира, к которому привязаны несколько касс ККМ, зарегистрированных на разные организации.

Список всех касс ККМ предприятия можно открыть по ссылке НСИ - Предприятие - Кассы ККМ. Кроме того, по ссылке Кассы ККМ из панели навигации карточки магазина (организации) можно открыть список касс ККМ данного магазина (организации). Для удобства работы Пользователь может отфильтровать список касс ККМ по магазину, выбрав нужное значение в поле Магазин в шапке списка. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные кассы ККМ, задавая поиск по значениям различных реквизитов.

Чтобы добавить кассу ККМ, нужно нажать кнопку Создать в списке касс ККМ и выбрать из выпадающего списка тип кассы:

  • Фискальный регистратор (ФР) или АСПД - используется, если к рабочему месту кассира подключено устройство для печати чеков, фискальный регистратор или АСПД (автоматизированная система печати документов). Предусмотрена возможность указания шаблона чеков при продаже и возврате товаров. Шаблоны чеков предварительно настраиваются в разделе Администрирование - Печатные формы, отчеты и обработки - Шаблоны этикеток, ценников и чеков ККМ.

  • ККМ без подключения оборудования - используется при применении автономного кассового аппарата или при оформлении продаж без кассового чека.

  • ККМ Offline - используется при продажах на ККМ в режиме Offline. В этом случае в информационную базу загружаются итоговые кассовые отчеты о продажах, оформленных в кассовой программе. Для ККМ этого типа необходимо указать соответствия видам оплаты при загрузке данных из ККМ offline.

Далее пользователь вводит организацию и магазин, к которой относится касса, указывает рабочее место (компьютер), на котором будет работать кассир, и вводит наименование кассы ККМ.

Важно, что рабочее место создается для каждого сочетания компьютер + пользователь операционной системы. Рабочие места также будут различаться при запуске программы в разных веб-браузерах на одной клиентской машине. Для типов оборудования Фискальный регистратор или АСПД и ККМ Offline можно указать оборудование соответствующего типа, подключенное к указанному рабочему месту. Для документов, распечатываемых с помощью узколенточного печатающего устройства, можно выбрать шаблон чека ККМ и указать ширину ленты.

Важно! При установленном ручном режиме формирования чеков нефискальные чеки печатаются не автоматически, а только при нажатии клавиши Печать чека в интерфейсе кассира (РМК).

Оплата платежными картами

Для того чтобы на Кассе ККМ была возможность оплаты платежными картами, необходимо подключть к кассе ККМ эквайринговый терминал. Для добавления эквайрингового теминала используется команда Перейти -Терминалы. Эквайринговые терминалы могут быть использованы с возможносью подключения торгового оборудования (платежного терминала) или без подключения торгового обрудования. Если торговое обрудование используется, то предварительно эквайринговый терминал должен быть добавлен в список торгового обрудования (Администрирование - Подключаемое обрудование).

Распределение выручки по секциям на ФР

В программе предусмотрена возможность настройки распределения выручки по секциям фискального регистратора. Администратор выбирает в поле Распределение выручки по секциям ФР нужный вариант из списка предварительно выполненных настроек.

В настройке предоставляется возможность разделить выручку по направлениям торговли:

· полученную при оформлении расходных и приходных кассовых ордеров;

· полученную при пробитии чеков розничным покупателям. В последнем случае предоставляется возможность разделения выручки по видам номенклатуры. В отдельных моделях фискальных регистраторов поддерживается до 30 секций.
