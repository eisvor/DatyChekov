Контрагенты

Справочник Контрагенты предназначен для введения и хранения информации о юридических лицах и частных предпринимателях, с которыми предприятие контактирует, осуществляя свою деятельность.

Иерархию справочника можно формировать так, как удобно для работы предприятия. Например, можно выделить следующие группы контрагентов:

  • Поставщики - контрагенты, которые поставляют свой товар непосредственно в магазины.

  • Покупатели - осуществляют закупки мелким оптом непосредственно в магазинах.

  • Банки - предоставляют услуги эквайринга в магазинах торговой сети.

Иерархия может быть многоуровневой. Например, внутри группы Поставщики можно выделить подгруппы поставщиков определенных видов товара или поставщиков с определенными условиями поставки.

Список контрагентов можно открыть по ссылке НСИ – Контрагенты. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные элементы справочника, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.

Заполнение карточки контрагента

В карточке Контрагента фиксируется необходимая для работы информация. С помощью флажков определяется тип контрагента: поставщик или покупатель.

Вводится наименование контрагента, как правило, соответствующее краткому наименованию его организации и указывается его правовой статус:

  • юридическое лицо,

  • физическое лицо,

  • юридическое лицо за пределами РК,

  • индивидуальный предприниматель.

Для юридических лиц можно указать полное наименование организации контрагента и ввести БИН и Код по ОКПО, если юридическое лицо зарегистрировано на территории Республики Казахстан. 

Для физических лиц указывается фамилия, имя и отчество физического лица, ИИН, Документ, удостоверяющий его личность и Код ОКПО, если физическое лицо зарегистрировано в качестве индивидуального предпринимателя.

В карточку можно также внести контактную информацию (адреса, телефоны) в соответствии с настройками видов контактной информации для контрагентов. Используя команду Добавить можно добавить дополнительные данные, например еще один телефон или дополнительный адрес электронной почты.

Чтобы ввести или изменить значения дополнительных сведений для контрагента, нужно нажать пиктограмму Открыть дополнительные сведения в командной панели списка или карточки выбранного контрагента.
Состав контактной информации и дополнительных сведений может редактировать пользователь, имеющий права администратора (раздел Администрирование - Общие настройки).

Переходы к дополнительной информации

Из панели навигации карточки контрагента можно открыть связанные с контрагентом дополнительные данные:

  • Банковские счета, сведения о которых будут использоваться при выписке документов.

  • Дисконтные карты, выданные контрагенту.

  • Номенклатура контрагентов (список номенклатуры, которую может поставить поставщик).

  • Поставляемая номенклатура (список номенклатуры, по которой были зарегистрированы поставки в магазины).

  • Поставщики номенклатуры.

  • Договоры по контрагенту.

Для контрагента- поставщика можно указать график оплаты для заполнения этапов оплат по умолчанию при оформлении поставки товаров ( в заказах поставшику или в документах поступления). Если у контрагента не выбран ни один график оплаты, то заполнение будет выполнено по предустановленному варианту - полная оплата наличными в день поставки.
