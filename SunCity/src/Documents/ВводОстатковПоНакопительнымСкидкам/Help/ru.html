Ввод остатков по накопительным скидкам

Документ «Ввод остатков по накопительным скидкам» предназначен для ввода данных о накоплениях покупателей для обслуживания накопительных скидок. Ввод остатков можно производить как в разрезе дисконтных карт, так и по номенклатуре.

Список документов открывается Администрирование - Первоначальное заполнение ИБ - Ввод остатков по накопительным скидкам.

Документ Ввод остатков денежных средств создается с помощью нажатия кнопки Создать.

В табличную часть документа с помощью кнопки Добавить вводится информация о дисконтной карте, (номенклатуре) и сумме. После того как остатки сформированы, нажмите Провести и закрыть.
