Отчет Комитенту о списании.

Товар комитента может быть списан. За списанные товары перед комитентом отчитываются отдельно. Для этих целей предназначен документ Отчет комитенту о списании.

Список документов Отчет Комитенту о списании открывается по ссылке Закупки - Комиссионная торговля – Отчеты комитенту. Для удобства работы Пользователь может отфильтровать список по комитенту, магазину, кассе, организации, ответственному, выбрав нужные значения в соответствующих полях шапки списка.

Новую запись Отчеты комитенту о списании можно добавить с помощью кнопки Создать (Отчет Комитенту о продажах) в командной панели списка. Вначале необходимо заполнить шапку документа, а затем добавить товары с помощью команды Добавить новый элемент, Найти по штрихкоду/магнитному коду или с помощью сервиса Заполнить (Заполнить по результатам продаж/Подобрать товары по результатам продаж).
