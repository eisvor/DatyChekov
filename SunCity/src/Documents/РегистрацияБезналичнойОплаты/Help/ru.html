Безналичная оплата

Документ предназначен для регистрации расчетов с контрагентами по безналичной форме оплаты.

Список документов открывается Финансы – Денежные средства – Безналичная оплата. Для удобства работы Пользователь может отфильтровать список по контрагенту, магазину, операции. С помощью команды Ctrl+F можно отфильтровать список по любой комбинации символов. С помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще.

 

Документ оформляется по ссылке Финансы - Денежные средства – Безналичная оплата – Создать в командной панели.

Оплата поставщику

При создании вручную в документе необходимо указать магазин, в который оплачивается поставка, организацию, от имени которой проводится платеж и контрагент, которому перечислены денежные средства.

В табличной части документа указываются документы-основания для проведения платежа. С помощью команды Подобрать не оплаченные... открывается список документов Заказ поставщику или Поступление товаров от данного поставщика, по которым выполнены еще не все запланированные безналичные выплаты. Отмеченные документы можно перенести в формируемый документ.

Возврат ДС от поставщика

При создании в документе необходимо указать магазин, в который возвращаются ДС, организацию и контрагента, который возвращает ДС.

В табличной части указываются документы-основания для проведения платежа. С помощью команды Подобрать не оплаченные... открывается список документов Заказ поставщику или Поступление товаров от данного поставщика, по которым выполнены еще не все запланированные безналичные выплаты. Отмеченные документы можно перенести в формируемый документ.
