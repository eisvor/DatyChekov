Уведомления о выводе из оборота

Уведомление о выводе из оборота - один из документов системы ИС МПТ, используемый при инвентаризации остатков промаркированной продукции. Также данный документ может применяться в некоторых неторговых операциях, в ходе которых товар списывается с баланса организации. Уведомление позволяет вывести из оборота коды маркировки по причинам, отличным от реализации розничному потребителю.

В целом вывод кодов маркировки из оборота возможен по нескольким основным причинам:

  • при розничной реализации промаркированного товара;

  • при использовании промаркированного товара для иных нужд, не подразумевающих продажу;

  • при повреждении/утери физической марки и/или товара, приводящей к невозможности дальнейшей реализации продукции.

    • В первом случае, т.е. при розничной реализации, все взаимодействие с ИС МПТ происходит на стороне ОФД. От пользователя в 1С в этом случае требуется выполнить заполнение стандартного чека продажи и с помощью сканера штрихкодов указать в нем коды маркировки с упаковок реализованного товара. После пробития чека информация о маркировке будет передана в ОФД, а затем в ИС МПТ, где произойдет автоматическое списание марок с учета.

    Во всех остальных случаях для списания кодов маркировки необходимо оформлять Уведомление о выводе из оборота с указанием причины списания. При заполнении документа нужно обратить внимание, что для некоторых товарных групп на стороне ИС МПТ установлено ограничение на возможный список причин выбытия. При этом причина указывается одна на весь список товаров документа. Состав товаров в Уведомлении должен быть монотоварным, т.е. не допускается смешение в одном документе разных товарных групп или товаров одной группы, имеющих различные GTIN.

    В некоторых случаях допускается оформление Уведомления о выводе из оборота с указанием причины списания, относящейся к реализации, например, по причине Вывод из оборота при интернет торговле. Такой сценарий может применяться в ситуациях, когда невозможно указать код маркировки непосредственно в чеке в момент его пробития: покупатель оплачивает товар и получает чек, а отгрузка и, соответственно, определение списываемого кода маркировки, происходит позднее.

    Заполненный документ следует отправить в ИС МПТ. После успешного прохождения Уведомлением всех проверок на стороне сервера указанные в нем коды маркировки будут списаны с баланса организации и выведены из оборота.
