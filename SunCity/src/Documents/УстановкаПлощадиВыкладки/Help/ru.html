Установка площади выкладки

Документ предназначен для фиксации размеров выкладки по товарным группам. Справочник Товарные группы размещен разделе НСИ - Номенклатура и используется, кроме оценки эффективности использования торговых площадей, для настройки размещения товаров по отделам торгового зала при поступлении в магазин (Закупки - Сервис - Настройка распределения товаров). Расширенный поиск (Alt+F), Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов. Дополнительные команды доступны через кнопку Еще.

Документ открывается по ссылке Маркетинг – Мерчандайзинг - Установка площади выкладки. Необходимо выбрать магазин, дату начала, и площади выкладки.

Анализ использования площади выкладки можно провести с помощью отчета Использование площади торгового пространства (Маркетинг - Отчеты по маркетингу).

Примечание. В качестве площади торгового пространства должна использоваться сопоставимая площадь экспозиции. Например, для выкладки на полках и вывески на крючках - это занятая торговой группой площадь полочного пространства или перфорированного стенда, при использовании прилавков и вешал - площадь пола торгового зала, занятая оборудованием. Товарные группы должны быть достаточно крупными, чтобы занимать устойчивое, неизменное длительный период времени пространство выкладки.
