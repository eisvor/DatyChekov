Настройки обмена с СУЗ

Настройки обмена с СУЗ позволяют определить, какое устройство выполняет обмен данными с зарегистрированной Станцией управления заказами (СУЗ).

Устройство для генерации серийных номеров предварительно регистрируется в личном кабинете ИС МПТ - СУЗ в разделе Устройства, где ему присваивается Токен - идентификатор подключения. Данный идентификатор необходимо указать в 1С для дальнейшей идентификации подключения к СУЗ по API.

В форме настроек обмена также указывается СУЗ - Станция управления заказами. Это серверное приложение, предоставляющее участнику оборота товаров, подлежащих обязательной маркировке, интерфейс для работы с генератором кодов маркировки и позволяющее управлять заданиями на эмиссию, нанесение и агрегирование кодов маркировки. Данные станций хранятся в отдельном справочнике Станции управления заказами.

Данные Организации определяют ИИН/БИН зарегистрированного в системе участника оборота маркированных товаров.
