Форма настройки обработчика драйвера "Контрольно-кассовая машина Webkassa 1.0"

Форма предназначена для настройки параметров работы контрольно-кассовых машин Webkassa с функцией передачи данных фискальных регистраторов, поддерживаемых драйвером «Webkassa: Фискальные регистраторы». Форма позволяет настраивать следующие параметры устройства:

  • Адрес сервиса
    URL адрес сервиса.
  • Логин
    Логин пользователя для ККМ Webkassa.
  • Пароль
    Пароль пользователя для ККМ Webkassa.
  • Автоматическая печать чека
    В случае установки настройки при пробитии чека будет автоматически произведена его печать на выбранном.
  • Ширина ленты
    Ширина ленты используемая при печати чеков.

После изменения значений параметров следует нажать кнопку "Записать и закрыть" для того, чтобы сохранить внесённые изменения, или закрыть окно для того, чтобы отказаться от их сохранения.
