Монитор показателей

Монитор показателей - отчет, в котором можно вывести различную информацию о деятельности магазина в одном месте. Доступность отчета включается настройкой Администрирование – Организации и финансы – Финансы - Использовать монитор показателей магазина.

Отчет можно добавить на начальную страницу программы.

Отчет доступен для пользователей с полными правами и для пользователей, которые включены в группы доступа «Управляющий», «Маркетолог».

Отчет доступен по ссылке Продажи – Отчеты по продажам – Показатели эффективности – Монитор показателей.

Так как отчет представляет собой набор отчетов, каждый из них можно представить в виде графика (настройки вызываются нажатием кнопки в виде шестеренки):

· График

· График по шагам

· График с накоплением

· График с областями

· График с областями и накоплением

· Гистограмма

· Гистограмма с накоплением Гистограмма объемная Гистограмма горизонтальная Круговая

· Круговая объемная Измерительная

с выбором палитры цветов, а также от настройки зависит будет ли выводиться линия тренда для показателей (доступно не для всех типов отображения).

Настройка определяет в разрезе чего будут формироваться данные:

· Разрез аналитики для формирования отчета. Изменение данного свойства применимо не ко всем видам отчетов.

· Периодичность расчета значений показателей. При большом периоде отчета не рекомендуется задавать маленькую периодичность. Это влияет на скорость его формирования.

· Масштаб показателя (коэффициент масштабирования значений)

· Точность расчета дробной части (количество знаков после запятой.)

· Количество диаграмм в ряду - количество графиков, выводимых в одном ряду. Выбор значений от 1 до 4.

· Период автообновления - частота автообновления данных отчета.

При выборе магазинов, для которых будут формироваться данные, если у пользователя ограничены права и заданы ограничения доступа по магазину, то отчет сформирует данные исходя из ограничений для пользователя.

Настройки отчета сохраняются для каждого пользователя индивидуально. При необходимости можно установить стандартные настройки, путем сброса (команда Еще – Стандартные настройки).
