Отчет график работы сотрудников

Аналитический отчет План/Факт по рабочему времени сотрудников(Персонал – Отчеты по персоналу – Сотрудники) предназначен для анализа фактически проработанного времени в сравнении с планом. Для каждого сотрудника можно сформировать отчет-расшифровку, уточняющий отклонения фактически проработанного времени от планового.
