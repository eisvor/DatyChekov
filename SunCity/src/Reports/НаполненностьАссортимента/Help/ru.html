Отчет “Наполненность ассортимента”

Отчет используется для создания данных по текущей наполненности ассортиментом. Открывается по ссылке Маркетинг – Отчеты по маркетинге – Ассортимент (должна быть включена опция - Администрирование - Маркетинг - Ассортимент - Квоты ассортимента).

В отчет отображаются данные об ассортименте магазинов на заданный пользователем момент времени. Для каждой номенклатурной записи, входящей в ассортимент, отображается ее роль в ассортименте, стадия жизненного цикла записи в ассортименте определенного формата и вид розничной цены. Данные в отчете группируются по форматам магазинов, видам номенклатуры и категориям. Для каждого уровня группировки отображается количество разных номенклатурных записей, присутствующих на заданный момент в ассортименте.
