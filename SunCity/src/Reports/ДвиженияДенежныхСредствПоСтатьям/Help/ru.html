Оборот денежных средств по статьям ДДС

В отчете показывается информация движении денежных средств в разрезе организаций и статей движения денежных средств. Отчет имеет несколько варианта настройки:

Оборот денежных средств по статьям ДДС за период - настройку удобно использовать для просмотра оборотов за короткий интервал времени. Отчет формируется в разрезе Организаций, Вида денежных средств (наличные\безналичные) и Статей ДДС;

Оборот денежных средств по статьям ДДС за период (по месяцам) - удобно использовать для просмотра оборотов за длительный промежуток времени. Обороты суммируются по месяцам и выводятся в колонки отдельно для приходных и расходных операций за каждый месяц, вошедший в период отчета.

Оборот денежных средств по статьям ДДС (в разрезе магазинов) - Оборот денежных средств выводится в разрезе Магазинов, Вида денежных средств (наличные\безналичные) и Статей ДДС;


Отчет открывается по ссылке Финансы – Отчеты по финансам – Денежные средства – Оборот наличных ДС по статьям ДДС.
