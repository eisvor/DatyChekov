Исполнительность поставщиков

Отчет Исполнительность поставщиков предназначен для анализа расхождений между заказами и поставками товаров, которые регистрируются в системе документами Заказ поставщику и Поступление товаров. Отчет открывается по ссылке Закупки - Отчеты по запасам и закупкам - Работа с поставщиками - Исполнительность поставщиков.

Отчет формируется по документам заказа, попадающим в период отчета. В качестве показателей исполнительности поставщика по каждому документу заказа рассчитываются расхождения с соответствующим поступлением по следующим данным:

· по количеству позиций,

· по общему количеству товара,

· по сумме,

· по срокам.

Учитываются отклонения как в большую, так и в меньшую сторону:суммируются модули расхождений. Рассчитываются относительные расхождения, как отношения абсолютных отклонений к показателям заказа.
