Предназначена для ввода населенного пункта по детальному шаблону.

Открывается из шаблона ввода адреса в поле Город, населенный пункт по кнопке Выбрать.

Ввод населенного пункта

  • В поле Субъект РК необходимо ввести наименование субъекта Республики Казахстана, к которому относится населенный пункт.
  • Если адресный классификатор для этого субъекта Республики Казахстан не заполнен, будет выведено сообщение.
  • Поля Регион, Район, Город, Нас.пункт и т.д. предназначены для ввода частей адреса. Поля, которые должны быть заполнены, зависят от адреса. Например, в некоторых адресах может не быть района, в других - города или населенного пункта. Если выбрано устаревшее наименование, программа выводит сообщение. Подтвердите свой выбор, нажав кнопку Да для продолжения ввода.
  • После окончания ввода города или населенного пункта нажмите ОК для продолжения ввода адреса.
