Обработка «Загрузка данных из файла»

Внешняя обработка «Загрузка данных из файла» предназначена для загрузки данных из внешних файлов (.xls) в справочник «Номенклатура» и документ «Установка цен номенклатуры».

После открытия обработки на закладке «Источник данных» по команде «Открыть файл» необходимо выбрать файл источника данных. После чтения данные файла отобразятся в табличном документе «Источник данных» формы обработки.

Если верхние строки открытого файла (заголовки прайс-листов, заголовки колонок таблиц) не должны обрабатываться при загрузке, то необходимо заполнить поле «Первая строка данных источника».

При выборе приемника данных заполнится таблица реквизитов приемника. При помощи кнопок «Вверх» и «Вниз» или перетаскивая мышью, необходимо совместить реквизит с колонкой содержащей необходимые данные. Если реквизит не нужно загружать или отсутствуют данные для его заполнения, то с реквизита необходимо снять пометку.

Если при загрузке не удалось найти значение реквизита, то реквизит можно заполнить значением по умолчанию, заполнив поле «Значение по умолчанию» напротив имени реквизита.

Если с реквизита снята пометка, то реквизит, безусловно, заполнится значением по умолчанию.

При загрузке, при необходимости, указываются поля поиска номенклуры уже существующей в базе данных. Полями поиска могут быть Код, Наименование, Артикул или Штрихкод номенклатуры. Если номенклатура найдена по ключевым полям, то карточка номенклатуры будет обновлена данными из источника данных. Если поля поиска не указаны, то поиск производиться не будет, будет создана новая карточка номенклатуры.

Если включена функциональная опция «Использовать дополнительные реквизиты и сведения» то поддерживается возможность загружать данные в дополнительные реквизиты и сведения номенклатуры и характеристик номенклатуры.

При загрузке в документ "Установка цен" обязательно указать хотя бы один из ключевых реквизитов поиска номенклатуры: Код, Наименование, Артикул или Штрихкод.

Перед загрузкой данных можно произвести проверку на закладке «Проверка данных», на которой выводятся ошибки поиска данных, а также не заполненности обязательных для заполнения реквизитов.
