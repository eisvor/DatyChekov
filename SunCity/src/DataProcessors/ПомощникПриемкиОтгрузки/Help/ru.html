Распределение товаров по складам

Сервис Распределение приемки(отгрузки) товаров по складам можно вызвать нажатием кнопки Распределить по складам из командной панели документов Поступление товаров и Реализация товаров или из командной панели списка этих документов, если для соответствующих хозяйственных операций настроена ордерная схема.

Также этот сервис можно вызвать из документа Перемещение товаров и из списка перемещений. Оформление перемещения товаров всегда производится по ордерной схеме.

Распределение товаров по складам магазина

По умолчанию строки табличной части, в том числе поля Склад и Количество (факт), заполняются из документа-основания.

Информация о складах может быть заполнена вручную для каждой строки из справочника Склады или автоматически. Для автоматического заполнения нужно нажать кнопку Заполнить склады и выбрать вариант заполнения данных:

По правилам распределения - товары должны поступить в торговые залы в соответствии с настройкой распределения по номенклатурным группам.

· По основанию - используется для восстановления значения склада из документа-основания.

· Из справочника – если весь товар должен поступить на склад, отличный от указанного в документе-основании, то нужно выбрать данный склад из справочника складов.

Те же способы заполнения складов доступны только для выбранных строк.

Фактическое количество приходуемого на указанные склады товара также можно изменить вручную или групповым заполнением. Для группового заполнения нужно нажать кнопку Заполнить количество. Доступны следующие варианты заполнения количества:

· По основанию - для восстановления количества указанного в документе-основании.

· Фиксированным значением - для всех строк документа устанавливается введенное пользователем значение. Это может быть полезно, например, при отборе образцов поступившего товара для витрины.

По окончании ввода информации пользователь нажимает кнопку Сформировать приходные (расходные) ордера. При этом будут сформированы Приходные ордера ( Расходные ордера) на указанные в таблице позиции номенклатуры с фактическим количеством на указанные склады.
Если одна и та же позиция должна быть размещена на несколько складов обработку нужно повторить. При повторном вызове сервиса табличная часть заполнится нераспределенными позициями номенклатуры с оставшимся количеством.
