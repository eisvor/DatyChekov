Организации и финансы

Панель предназначена для управления параметрами организации и учета денежных средств. Панель открывается по ссылке Администрирование – Организации и финансы.

Организации

· Несколько организаций - Возможность ведения нескольких организаций.

· Список организаций – Справочник организаций в БД.

· Передачи товаров между организациями - Осуществление передачи товаров между собственными организациями.

Финансы

· Выплата заработной платы в магазинах - Возможность выплаты зарплаты сотрудникам из операционной кассы магазина.

· Кассовая книга - Ведение отчетности по операциям с наличными денежными средствами, позволяет создать и распечатать листы кассовой книги по операциям с наличными денежными средствами.

· По магазинам - Ведение кассовой книги в разрезе магазинов (подразделений).

· Заявки на расходование денежных средств – флажок доступен, если используется хотя бы один обмен с конфигурацией «Управление торговлей». При использовании этого режима выплаты из операционной кассы магазина возможны только по утвержденной в управляющей системе заявке.

· Статьи движения денежных средств - Справочник используется для детализации учета потока денежных средств. Помимо предусмотренных в программе предопределенных элементов для каждой хозяйственной операции с денежными средствами пользователь может ввести дополнительные статьи ДДС, которые указываются в каждом кассовом ордере. Это позволяет получить отчеты с необходимой пользователю детализацией.

· Агентские платежи и разделение выручки - Возможность ведения учета по операциям, оказываемым платежными агентами, по приему денежных средств от покупателей. При включении опции автоматически будет обновлен командный интерфейс, и в панели навигации раздела "Финансы" появятся "Договоры платежных агентов", которые будут доступны для выбора в номенклатуре и финансовых документах.

· Рассчитывать премии в магазинах - Включает возможность расчета премий в магазинах.

· Использовать монитор показателей магазина - Возможность формирования монитора основных показателей магазина.
