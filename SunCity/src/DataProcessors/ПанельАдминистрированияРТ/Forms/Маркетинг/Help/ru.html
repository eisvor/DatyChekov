Маркетинг

Управление настройками цен. скидок и наценок, использования ассортиментной политики. Ценообразование. Панель открывается по ссылке Администрирование – Маркетинг.

Ценообразование

· Ценовые группы - Ведение перечня групп номенклатуры, объединяющих товарные позиции, для которых применяются одинаковые правила расчета цены и расчета скидок.

· Применение цен – флажок, позволяет вводить в действие розничные цены через некоторый промежуток времени после получения с помощью обмена назначенных в офисе цен. Этот промежуток времени технологически необходим для замены ценников и этикеток.

Скидки, наценки, ограничения

· Ручные скидки в продажах – Возможность назначения ручных скидок при оформлении продаж.

· Скидки, наценки и ограничения продаж - Условия для автоматического расчета и назначения скидок, наценок и ограничений продаж.

· Бонусные программы лояльности - Учет, начисление и списание бонусных баллов.

· Использовать неуникальные коды информационных карт - Отключение контроля уникальности кодов магнитных карт и штрихкодов информационных карт.

· Использовать запрет розничной продажи алкоголя - Включается возможность установки запретов продаж по времени и сегментам номенклатуры для документов розничной продажи (Чек ККМ и РМК).

Дисконтный сервер

· Подключаться к дисконтному серверу - Возможность получения данных о накоплениях по дисконтным картам on-line из главного узла РИБ

· Использовать очередь передачи дисконтному серверу - Определяет режим отправки данных для обновления базы дисконтного сервера: немедленная или отложенная отправка

· Настройка подключения к дисконтному серверу - Настройка адреса web-сервиса дисконтного сервера, указание имени пользователя и пароля для подключения к БД.

· Протокол передачи продаж дисконтному серверу - Сведения обо всех документах, отправленных или ожидающих отправки дисконтному серверу

Ассортимент

· Управление ассортиментом – флажок, добавляет функциональность управления ассортиментом сети магазинов. При этом пользователю становятся доступны пункты подраздела Ассортимент раздела Маркетинг для ведения в системе форматов магазинов, категорий товаров, ассортиментов магазинов по форматам, а также квот ассортимента. Помощник начального формирования ассортимента позволяет быстро сформировать текущий ассортимент форматов магазинов по заданным пользователем правилам

· Помощник начального Формирования ассортимента - Создание форматов магазинов, документов ввода в ассортимент.

· Квоты ассортимента и товарные категории. Использование товарных категорий, как расширения дерева видов номенклатуры Установка квот ассортимента по категориям. Включение дополнительных отчетов по исполнению ассортимента и по продажам в разрезе категорий.

· Устанавливать виды цен в ассортименте. Виды цен, указанные в правилах ценообразования будут игнорироваться, а сведения о виде цен для каждого товара нужно задавать в документе изменения ассортимента.

С помощью последних двух опций пользователь может постепенно расширять функции управления ассортиментом. Если обе опции отключены, то маркетологу доступны функции управления жизненным циклом товара в розничной сети: определение формата магазина с помощью справочника Форматы магазинов и документы ввода номенклатуры в ассортимент формата магазина (Ассортимент). Ассортимент магазина формируется в разрезе номенклатурных позиций. После проведения документа все характеристики номенклатурной позиции считаются введенными в ассортимент. Это следует учитывать, при формировании правил ведения справочника номенклатуры. Количество товарных позиций в разрезе видов номенклатуры, их роль в ассортименте магазина и стадию жизненного цикла можно просмотреть с помощью отчета Текущая наполненность ассортимента.

Настройка Квоты ассортимента и товарные категории позволяет расширить справочник видов номенклатуры товарными категориями и для каждой товарной категории квотировать количество товарных позиций на полках магазина. При включении настройки маркетолог сможет вести справочник Категории, расширяющий Виды номенклатуры и создавать документ Квоты ассортимента, определяя количество позиций в данной категории, которое должно быть на полках магазинов определенного формата. Исполнение плановых показателей можно контролировать отчетами Квоты ассортимента и Анализ исполнения ассортимента. Доступна также группа отчетов, анализирующих в разрезе товарных категорий показатели по основным операциям с товаром: продажи, возвраты от покупателей и поставщикам, списания, а также оценка склада магазина в разных ценах.

Настройка Устанавливать виды цен в ассортименте, позволяет изменить способ розничного ценообразования в магазине. Виды цен, указанные в правиле ценообразования для выбранного магазина не будут использоваться для определения цены розничной продажи. Маркетолог сможет назначить Вид цен продажи товаров непосредственно в документе Ассортимент, что удобно для торговых предприятий, продающих преимущественно сезонный товар.
