Обмен с банком

Обработка предназначена для автоматизированного создания документов "Регистрация безналичной оплаты" на основании банковских выписок, полученных через специализированную программу типа "Клиент-Банк".

Неподдерживаемые операции

По умолчанию ряд операций выписки отображается справочно:

  • Перевод между счетами;
  • Снятие наличных;
  • Инкассация;
  • Поступление оплат по платежным картам (эквайринг);
  • Платежи в бюджет;

Настройка обмена с банком

Настройка производится из формы обработки для каждого расчетного счета организаций предприятия. Если в организации открыто несколько расчетных счетов, то необходимо сделать несколько настроек. При выборе банковского счета будет автоматически установлена нужная настройка.
