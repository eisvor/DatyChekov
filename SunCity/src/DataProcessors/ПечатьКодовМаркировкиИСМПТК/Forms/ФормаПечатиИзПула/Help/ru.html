Печать кодов маркировки

Форма Печать кодов маркировки используется для уточнения данных при печати марок из Пула кодов. В этой форме устанавливается шаблон этикетки, который будет использоваться при печати кодов маркировки, а также уточняется какое количество этикеток необходимо распечатать.

Настройка Расположение этикеток на листе позволяет определить способ размещения данных при печати:

  • Замостить - при выборе этого режима на листе будет размещаться максимально возможное количество этикеток, они будут располагаться край к краю, одна за другой по всей имеющейся площади листа;

  • На отдельных листах - при выборе этого режима каждая этикетка будет размещаться на отдельном листе.

    • В графе Количество (в пуле) отображается информация о доступном для печати количестве кодов маркировки по комбинации "Заказ на эмиссию + GTIN" в Пуле кодов. Поле недоступно для редактирования и служит для информирования пользователя.

    В графе Количество (на печать) указывается какое количество кодов маркировки (уникальных марок, а не экземпляров одного кода) по комбинации "Заказ на эмиссию + GTIN" необходимо распечатать. Данное значение заполняется вручную. При этом нельзя указать большее количество, чем указано в графе Количество (в пуле).

    Команда Подобрать номенклатуру позволяет заполнить сведения о товарах перед печатью этикеток кодов маркировки. Это может потребоваться, если при оформлении Заказа на эмиссию данные номенклатуры не были указаны в документе (они не являются обязательными для заполнения). В этом случае подбор номенклатуры в форме печати позволяет заполнить соответствующую информацию в полях макета печати, не изменяя при этом сведения в Пуле кодов и в исходных документах.

    Команда Заполнить шаблоны в группе команд Заполнить для всех позволяет установить единый шаблон печати во всех строках таблицы. Также возможно вручную выбрать подходящий шаблон печати в каждой строке отдельно, в этом случае необходимо скорректировать значения в колонке Шаблон этикетки.

    Команда Заполнить количество в группе команд Заполнить для всех позволяет автоматически заполнить выбранным значением графу Количество (на печать) во всех строках таблицы. При этом если в какой-либо строке таблицы указанное пользователем количество для заполнения будет превышать имеющийся в Пуле свободный остаток марок, значение в ней изменено не будет.

    Команда Печать позволяет открыть форму предпросмотра сгенерированных по выбранным параметрам этикеток кодов маркировки.
