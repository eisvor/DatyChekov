РМК управляемый режим

Для оформления розничных продаж потоковым методом служит специализированный графический интерфейс рабочего места кассира. Интерфейс оптимизирован для создания списка товаров с помощью сканера штрихкода. Реализован также быстрый поиск по наиболее востребованным полям: артикул, наименование, код товара.

Интерфейс позволяет работать на кассовых местах, оборудованных как обычной компьютерной, так и программируемой клавиатурой, поддерживается работа с сенсорными экранами.

При входе в РМК открывается основное меню РМК, с помощью команд которого можно производить необходимые при работе кассира операции. Доступность команд основного меню может быть настроена администратором.
