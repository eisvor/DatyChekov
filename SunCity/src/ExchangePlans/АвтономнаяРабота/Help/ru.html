Обмен «Автономная работа»

Назначение

Обмен предназначен для автономной работы приложения Розница, редакция 2.3 без подключения к сети Интернет с возможностью синхронизации данными между Автономным рабочим местом (АРМ) и приложением в сети Интернет.

Общее описание

Обмен между АРМ и приложением в сети Интернет заключает в себе две основных функции:

  • Синхронизация с передачей обновлений конфигурации из приложения в интернете в автономное рабочее место.
  • Синхронизация данными в обе стороны.

Обмен позволяет создавать Автономные рабочие места в зависимости от назначения:

  • Синхронизация по варианту Автономный магазин, предназначен для ведения учета в удаленном магазине.
  • Синхронизация по варианту Автономное место кассира, предназначен для осуществления розничных продаж в автономном месте кассира.

Отправка данных по варианту Автономный магазин осуществляется с фильтром по Магазину.
Отправка данных по варианту Автономное место кассира осуществляется с фильтром по Магазину и Рабочему месту.

Дополнительно имеется возможность отправки в АРМ информативных данных по остаткам товаров по другим магазинам розничной сети.

Отправка данных в АРМ возможна как за весь период ведения учета, так и начиная с конкретной даты - Даты начала выгрузки.
Данные ранее даты начала выгрузки не отправляются, поэтому для варианта Автономный магазин есть возможность сформировать документ "Начальные остатки узла", который отправляется в АРМ для обеспечения корректности данных. Для варианта Автономное место кассира документ не формируется т.к. в АРМ такого типа учет остатков товаров не ведется.
